# Financial-Sales-forecasting-Project
End‑to‑end financial sales forecasting project using Python (ARIMA) and Power BI. Built interactive dashboards with KPIs, line charts, and error diagnostics. Achieved 81.64% forecast accuracy with MAPE of 18.36% and RMSE of 5.04M, enabling reliable financial planning.


## Project Overview
This project demonstrates how forecasting models can be combined with business‑ready dashboards to provide actionable insights for financial planning and resource allocation.  

The workflow includes:
- Data preparation and exploratory analysis in Python (Pandas, NumPy).  
- ARIMA modeling for monthly sales forecasting.  
- Export of forecasts and integration into Power BI.  
- Dashboard design with KPIs, line charts, error diagnostics, and yearly breakdowns.  
- Scaling correction applied to align forecast units with actuals.  

## Tools & Technologies
- Python: Pandas, Statsmodels (ARIMA modeling)  
- Power BI: DAX, Power Query (KPI calculations, dashboard design)  

## Key Metrics
- Forecast Accuracy: 81.64%  
- MAPE: 18.36%  
- RMSE: 5.04M  
- Total Actual Sales: 134M  
- Total Forecast Sales (scaled): 135.07M  

## Dashboard Highlights
- KPI cards for instant performance visibility  
- Line chart tracking actual vs forecast sales (2019–2024)  
- Error by Month chart for diagnostic insights  
- Yearly summary table with MAE and RMSE for transparency  

## Business Impact
- Improved forecast accuracy supports better financial planning and resource allocation  
- Transparent error metrics build stakeholder trust  
- Demonstrates ability to combine technical modeling with business storytelling in a recruiter‑ready format
